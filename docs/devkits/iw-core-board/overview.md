# iW-CORE-BOARD

A `iW-CORE-BOARD V1` foi criada para encurtar o caminho entre protótipo e aplicação final. Em vez de partir de uma placa genérica e reconstruir toda a base de produto, a ideia aqui é usar uma plataforma modular pronta para receber placas de aplicação da linha `IW-APPLICATION`.

![Placa iW-CORE-BOARD-V1](/img/iw-core-board/BOARD_VIEW.png)

---

## Destaques

- STM32H750 com Cortex-M7 a 480 MHz
- 32 MB de memória externa em XiP
- DMA2D e JPEG Codec para gráficos e imagem
- display integrado de 4.3" com touch capacitivo
- módulo Quectel Wi-Fi + Bluetooth
- PMIC, RTC, bateria e slot SD

---

## Recursos principais

### Interface e experiência

- display de 4.3" com resolução 480 x 272
- touch capacitivo para HMIs e produtos embarcados

### Processamento

- MCU de alta performance para aplicações mais exigentes
- aceleradores voltados a interfaces e processamento gráfico

### Conectividade e expansão

- módulo Wi-Fi + Bluetooth
- USB-C para debug serial
- USB FS
- dois conectores `IW-APPLICATION`

### Energia

- PMIC integrado
- suporte a bateria BL-5C ou JST
- RTC com CR1220
- botão dedicado de power on/off

---

## Aplicações sugeridas

- IHMs industriais
- produtos IoT mais completos
- equipamentos de campo com interface local
- sistemas embarcados profissionais com expansão modular

---

## Proposta da plataforma

A iW-CORE-BOARD existe para que a equipe foque no produto final mais cedo. Ela reduz etapas de infraestrutura de hardware e abre espaço para desenvolver funcionalidade, UX e integração sem perder semanas montando uma base do zero.

# iW-CORE-BOARD

A `iW-CORE-BOARD V1` foi criada para encurtar o caminho entre prototipo e aplicacao final. Em vez de partir de uma placa generica e reconstruir toda a base de produto, a ideia aqui e usar uma plataforma modular pronta para receber placas de aplicacao da linha `IW-APPLICATION`.

![Placa iW-CORE-BOARD-V1](/img/iw-core-board/BOARD_VIEW.png)

---

## Destaques

- STM32H750 com Cortex-M7 a 480 MHz
- 32 MB de memoria externa em XiP
- DMA2D e JPEG Codec para graficos e imagem
- display integrado de 4.3" com touch capacitivo
- modulo Quectel Wi-Fi + Bluetooth
- PMIC, RTC, bateria e slot SD

---

## Recursos principais

### Interface e experiencia

- display de 4.3" com resolucao 480 x 272
- touch capacitivo para HMIs e produtos embarcados

### Processamento

- MCU de alta performance para aplicacoes mais exigentes
- aceleradores voltados a interfaces e processamento grafico

### Conectividade e expansao

- modulo Wi-Fi + Bluetooth
- USB-C para debug serial
- USB FS
- dois conectores `IW-APPLICATION`

### Energia

- PMIC integrado
- suporte a bateria BL-5C ou JST
- RTC com CR1220
- botao dedicado de power on/off

---

## Aplicacoes sugeridas

- IHMs industriais
- produtos IoT mais completos
- equipamentos de campo com interface local
- sistemas embarcados profissionais com expansao modular

---

## Proposta da plataforma

A iW-CORE-BOARD existe para que a equipe foque no produto final mais cedo. Ela reduz etapas de infraestrutura de hardware e abre espaco para desenvolver funcionalidade, UX e integracao sem perder semanas montando uma base do zero.

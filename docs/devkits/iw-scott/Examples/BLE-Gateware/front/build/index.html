BLE Gateway Console

Provisione Wi-Fi e MQTT sem recompilar o firmware

Esta interface fica gravada na flash do ESP32-C3. Use-a para apontar o gateway para sua rede Wi-Fi e para o broker MQTT que vai receber os dados dos beacons.

Provisionamento

Fluxo rapido

  1. Conecte no access point do gateway e abra este painel.
  2. Preencha o SSID e a senha da rede local.
  3. Informe a URI do broker MQTT, o client ID e o topico de publicacao.
  4. Salve. O gateway mantem o BLE ativo e tenta subir o Wi-Fi em paralelo.
Observacao

O firmware permanece anunciando o painel pela rede AP do proprio gateway, entao voce continua com acesso local mesmo enquanto o STA negocia a conexao.

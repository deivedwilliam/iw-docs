# iW-DigitalAmplifier

A `iW-DigitalAmplifier` é uma breakout board voltada para projetos de áudio digital que precisam de boa potência, alta eficiência e integração simples com fontes de áudio embarcadas.

![Placa iW-DigitalAmplifier](/img/iw-digitalamplifier/BOARD_VIEW.png)

---

## Destaques

- CI `TAS5805`
- amplificador stereo classe D
- até 23 W por canal
- entrada digital via I2S ou TDM
- baixa dissipação térmica
- formato compacto para protótipos

---

## Aplicações ideais

- caixas de som sem fio
- smart speakers
- dispositivos IoT com áudio
- protótipos DIY de áudio digital

---

## Por que este módulo é útil

Quando o objetivo é validar rapidamente um produto com áudio, a iW-DigitalAmplifier evita perder tempo com o desenvolvimento de uma etapa de potência do zero. Ela ajuda a equipe a focar no DSP, na experiência do usuário e na integração com o restante do sistema.

# Development Boards

This section gathers the main board line from Indústrias William. The focus here is to document each board's purpose, hardware features, supported stacks and examples that help with day-to-day bench work.

---

## Current catalog

| Product | Platform | Highlight |
| --- | --- | --- |
| [iW-ST Starter Kit](/docs/devkits/iw-starter-kit) | STM32F030F4P6 | Compact entry point to explore the STM32 ecosystem |
| [iW-Scott DevKit](/docs/devkits/iw-scott) | ESP32-C3 | Compact devkit for edge, BLE, sensors and local interface |
| [iW-ESPMI-IND](/docs/devkits/iw-espmi-ind) | ESP32-S3 | Industrial platform with HDMI, RS485 and SD Card |
| [iW-CORE-BOARD](/docs/devkits/iw-core-board) | STM32H750 | Modular base for final products with HMI and expansion |
| [iW-Beacon](/docs/devkits/iw-beacon) | nRF52832 | BLE beacon with sensors and a nano-power timer for long battery life |
| [iW-AIoT-eXplorer](/docs/devkits/iw-aiot-explorer) | ESP32-S3R8 | AIoT platform with camera, microphones, touch display and LoRa |

# Examples

Even when the final firmware is not finished yet, the iW-ESPMI-IND pairs very well with a few quick-validation scenarios.

---

## HMI over HDMI

Build a local interface for:

- machine status
- parameter adjustment
- industrial dashboards
- maintenance screens

---

## Modbus RTU master

Use the integrated RS485 to:

- read sensors and instruments
- write setpoints
- validate network topologies
- build local supervision panels

---

## Logger with SD Card

Record locally:

- alarm history
- process trends
- operating parameters
- configuration files

---

## A powerful combination

The real strength of this board is precisely the combination of:

- local visual interface
- industrial communication
- embedded storage

This significantly reduces the amount of auxiliary hardware in the prototype phase.

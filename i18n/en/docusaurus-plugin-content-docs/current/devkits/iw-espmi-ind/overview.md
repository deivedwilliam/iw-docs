# iW-ESPMI-IND

The `iW-ESPMI-IND` is an industrial development board based on the **ESP32-S3**, created for projects that need to combine a visual interface, local storage and robust communication in an automation environment.

![iW-ESPMI-IND board](/img/iw-espmi-ind/BOARD_VIEW.png)

---

## Highlights

- ESP32-S3 for IoT and embedded computing
- 16 MB of Flash
- 8 MB of Octal PSRAM
- HDMI video output
- native RS485 for industrial protocols
- SD card slot
- wide power input up to 36 V

---

## Where this board shines

- industrial HMIs
- embedded dashboards
- gateways with a visual interface
- remote monitoring and control
- integration with industrial buses such as Modbus

---

## Compatibility

- Arduino IDE
- ESP-IDF

---

## Why it stands out

The combination of **ESP32-S3 + HDMI + RS485 + SD Card** makes the iW-ESPMI-IND a very versatile platform for industrial projects. It can handle everything from local graphical interfaces to field communication, without requiring a stack of extra boards to reach the first working prototype.

# iW-Scott DevKit

The `iW-Scott DevKit` was designed for edge computing, IoT and portable-device projects. It is a compact board with wireless connectivity, an on-board sensor and power features that help both with prototypes and with demos closer to the final product.

![iW-Scott board](/img/iw-scott/BOARD_VIEW.png)

---

## Highlights

- ESP32-C3 with Wi-Fi and Bluetooth 5 LE
- 8 MB or 16 MB of Flash, depending on the version
- Integrated LIS3DH accelerometer
- On-board RGB LED
- PT1502 PMIC with battery support
- U.FL connector for an external antenna

---

## Compatibility

- Arduino IDE
- ESP-IDF
- MicroPython
- PlatformIO

---

## Suggested applications

- low-power IoT
- home automation
- wearables and portable devices
- compact BLE gateways
- local interfaces with a small display

---

## What already exists in the project

This folder already contains practical examples for the board, including:

- visual control with the accelerometer
- a game with a display and a BLE joystick
- a BLE gateway for reading and publishing sensors

See the [Examples](/docs/devkits/iw-scott/examples) page for a summary of each case.

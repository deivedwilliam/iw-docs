# iW-CORE-BOARD

The `iW-CORE-BOARD V1` was created to shorten the path between prototype and final application. Instead of starting from a generic board and rebuilding the entire product base, the idea here is to use a modular platform ready to host application boards from the `IW-APPLICATION` line.

![iW-CORE-BOARD-V1 board](/img/iw-core-board/BOARD_VIEW.png)

---

## Highlights

- STM32H750 with Cortex-M7 at 480 MHz
- 32 MB of external XiP memory
- DMA2D and JPEG Codec for graphics and imaging
- integrated 4.3" display with capacitive touch
- Quectel Wi-Fi + Bluetooth module
- PMIC, RTC, battery and SD slot

---

## Main features

### Interface and experience

- 4.3" display with 480 x 272 resolution
- capacitive touch for HMIs and embedded products

### Processing

- high-performance MCU for more demanding applications
- accelerators aimed at interfaces and graphics processing

### Connectivity and expansion

- Wi-Fi + Bluetooth module
- USB-C for serial debug
- USB FS
- two `IW-APPLICATION` connectors

### Power

- integrated PMIC
- support for a BL-5C or JST battery
- RTC with CR1220
- dedicated power on/off button

---

## Suggested applications

- industrial HMIs
- more complete IoT products
- field equipment with a local interface
- professional embedded systems with modular expansion

---

## Platform proposition

The iW-CORE-BOARD exists so the team can focus on the final product sooner. It removes hardware-infrastructure steps and frees you up to develop functionality, UX and integration without spending weeks building a base from scratch.

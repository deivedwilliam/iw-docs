# Examples

Incluso cuando el firmware final aún no está cerrado, la iW-ESPMI-IND combina muy bien con algunos escenarios de validación rápida.

---

## HMI con HDMI

Monta una interfaz local para:

- estado de máquina
- ajuste de parámetros
- dashboards industriales
- pantallas de mantenimiento

---

## Maestro Modbus RTU

Usa el RS485 integrado para:

- leer sensores e instrumentos
- escribir setpoints
- validar topologías de red
- montar paneles de supervisión local

---

## Logger con tarjeta SD

Registra localmente:

- histórico de alarmas
- tendencias de proceso
- parámetros de operación
- archivos de configuración

---

## Una combinación poderosa

El gran diferencial de esta placa es justamente la combinación de:

- interfaz visual local
- comunicación industrial
- almacenamiento embebido

Esto reduce bastante la cantidad de hardware auxiliar en la fase de prototipo.

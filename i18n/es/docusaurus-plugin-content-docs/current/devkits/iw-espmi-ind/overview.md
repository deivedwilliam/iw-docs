# iW-ESPMI-IND

La `iW-ESPMI-IND` es una placa de desarrollo industrial basada en el **ESP32-S3**, creada para proyectos que necesitan combinar interfaz visual, almacenamiento local y comunicación robusta en un entorno de automatización.

![Placa iW-ESPMI-IND](/img/iw-espmi-ind/BOARD_VIEW.png)

---

## Destacados

- ESP32-S3 para IoT y computación embebida
- 16 MB de Flash
- 8 MB de Octal PSRAM
- salida de vídeo HDMI
- RS485 nativo para protocolos industriales
- ranura para tarjeta SD
- alimentación amplia de hasta 36 V

---

## Dónde destaca esta placa

- HMI industriales
- dashboards embebidos
- gateways con interfaz visual
- monitoreo y control remoto
- integración con buses industriales como Modbus

---

## Compatibilidad

- Arduino IDE
- ESP-IDF

---

## Por qué llama la atención

La combinación de **ESP32-S3 + HDMI + RS485 + tarjeta SD** convierte a la iW-ESPMI-IND en una plataforma muy versátil para proyectos industriales. Puede atender desde interfaces gráficas locales hasta comunicación de campo, sin exigir una pila de placas extra para llegar al primer prototipo funcional.

# iW-Scott DevKit

La `iW-Scott DevKit` fue diseñada para proyectos de edge computing, IoT y dispositivos portátiles. Es una placa compacta, con conectividad inalámbrica, sensor integrado y recursos de energía que ayudan tanto en prototipos como en demostraciones más cercanas al producto final.

![Placa iW-Scott](/img/iw-scott/BOARD_VIEW.png)

---

## Destacados

- ESP32-C3 con Wi-Fi y Bluetooth 5 LE
- 8 MB o 16 MB de Flash, según la versión
- Acelerómetro LIS3DH integrado
- LED RGB integrado
- PMIC PT1502 con soporte de batería
- Conector U.FL para antena externa

---

## Compatibilidad

- Arduino IDE
- ESP-IDF
- MicroPython
- PlatformIO

---

## Aplicaciones sugeridas

- IoT de bajo consumo
- automatización del hogar
- wearables y dispositivos portátiles
- gateways BLE compactos
- interfaces locales con pantalla pequeña

# iW-CORE-BOARD

La `iW-CORE-BOARD V1` fue creada para acortar el camino entre el prototipo y la aplicación final. En lugar de partir de una placa genérica y reconstruir toda la base de producto, la idea aquí es usar una plataforma modular lista para recibir placas de aplicación de la línea `IW-APPLICATION`.

![Placa iW-CORE-BOARD-V1](/img/iw-core-board/BOARD_VIEW.png)

---

## Destacados

- STM32H750 con Cortex-M7 a 480 MHz
- 32 MB de memoria externa en XiP
- DMA2D y JPEG Codec para gráficos e imagen
- pantalla integrada de 4.3" con touch capacitivo
- módulo Quectel Wi-Fi + Bluetooth
- PMIC, RTC, batería y ranura SD

---

## Características principales

### Interfaz y experiencia

- pantalla de 4.3" con resolución 480 x 272
- touch capacitivo para HMI y productos embebidos

### Procesamiento

- MCU de alto rendimiento para aplicaciones más exigentes
- aceleradores orientados a interfaces y procesamiento gráfico

### Conectividad y expansión

- módulo Wi-Fi + Bluetooth
- USB-C para debug serial
- USB FS
- dos conectores `IW-APPLICATION`

### Energía

- PMIC integrado
- soporte para batería BL-5C o JST
- RTC con CR1220
- botón dedicado de encendido/apagado

---

## Aplicaciones sugeridas

- HMI industriales
- productos IoT más completos
- equipos de campo con interfaz local
- sistemas embebidos profesionales con expansión modular

---

## Propuesta de la plataforma

La iW-CORE-BOARD existe para que el equipo se enfoque en el producto final antes. Reduce etapas de infraestructura de hardware y abre espacio para desarrollar funcionalidad, UX e integración sin perder semanas montando una base desde cero.

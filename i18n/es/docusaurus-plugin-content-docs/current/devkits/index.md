# Development Boards

Esta sección reúne la línea principal de placas de Indústrias William. El foco aquí es documentar el propósito de cada placa, las características de hardware, los stacks compatibles y los ejemplos que ayudan en el día a día de banco.

---

## Catálogo actual

| Producto | Plataforma | Destacado |
| --- | --- | --- |
| [iW-ST Starter Kit](/docs/devkits/iw-starter-kit) | STM32F030F4P6 | Punto de entrada compacto para explorar el ecosistema STM32 |
| [iW-Scott DevKit](/docs/devkits/iw-scott) | ESP32-C3 | Devkit compacta para edge, BLE, sensores e interfaz local |
| [iW-ESPMI-IND](/docs/devkits/iw-espmi-ind) | ESP32-S3 | Plataforma industrial con HDMI, RS485 y tarjeta SD |
| [iW-CORE-BOARD](/docs/devkits/iw-core-board) | STM32H750 | Base modular para productos finales con HMI y expansión |
| [iW-Beacon](/docs/devkits/iw-beacon) | nRF52832 | Beacon BLE con sensores y temporizador nano-power para larga autonomía |
| [iW-AIoT-eXplorer](/docs/devkits/iw-aiot-explorer) | ESP32-S3R8 | Plataforma AIoT con cámara, micrófonos, pantalla táctil y LoRa |
